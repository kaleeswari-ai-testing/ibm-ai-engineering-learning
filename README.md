# IBM AI Engineering – Learning & AI Testing Practice

## About
This repository contains my learning, labs, and projects from the IBM AI Engineering Professional Certificate.
I focus on understanding AI models and applying a testing mindset to validate model behavior, accuracy,
and reliability.

## Skills Covered
- Machine Learning & Deep Learning fundamentals
- Model validation and evaluation
- NLP & Generative AI basics
- Python for AI testing
- Data-driven testing concepts

## Repository Structure
- notebooks/        → Jupyter notebooks from labs
- datasets/         → Sample datasets used
- models/           → Trained models (if applicable)
- testing-notes/    → AI testing observations & test scenarios
- projects/         → Mini projects & capstone work

## Goal
To build hands-on experience in AI & ML testing and create a portfolio demonstrating practical
AI validation skills.
